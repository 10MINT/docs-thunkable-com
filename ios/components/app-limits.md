# App Limits

---

Due to popular demand, Thunkable has increased app size limits to 50 MB per app \(compared with 10 MB on Thunkable for Android\)

| Element | Limit |
| :--- | :--- |
| App size | 50 MB |
| Account size limit | 200 MB |
| Screens | No limit |
| Download | One app per phone |
