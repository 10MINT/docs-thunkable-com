# Photo Library \(aka Image Picker\) ![](/assets/photo-lib-ios-icon.png)

---

#### Functionality

| Property | Description |
| :--- | :--- |
| Select Photo \(`Photo`, `DidUserCancel`, `Error`\) | Opens the camera and saves the `Photo` after the user takes a picture |
